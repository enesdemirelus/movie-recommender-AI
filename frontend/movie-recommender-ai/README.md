## Movie Recommender AI (Frontend)

This folder contains the **Next.js** app (App Router) for the Movie Recommender UI.

### Running locally

From this directory:

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

### Environment variables

To enable `/api/get-movie-details`, create `.env.local`:

```bash
TMDB_API=your_tmdb_api_key_here
```

### API routes

- **`GET /api/get-movies`**: loads movies from `data/movies_meta.json` (supports `query` + `limit`)
- **`GET /api/recommend-movies?movie_id=...`**: returns 10 similar movies using `data/movie_vectors.json`
- **`GET /api/get-movie-details?movie_id=...`**: fetches extra details from TMDB (needs `TMDB_API`)

### Data

The app uses precomputed datasets committed in:
- `data/movies_meta.json`
- `data/movie_vectors.json`

These are generated by backend scripts (see `../../backend/README.md` and the root `README.md`).
